# 🛒 Toko Sembako ARNOF

Aplikasi Manajemen Toko Sembako modern yang dibangun menggunakan stack **React (Vite)** untuk Frontend dan **Node.js (Express)** dengan **Prisma ORM** untuk Backend.

## ✨ Fitur Utama
- **Dashboard Overview**: Ringkasan data barang, transaksi, dan pendapatan.
- **Manajemen Barang**: Inventaris stok barang lengkap dengan harga.
- **Manajemen Kategori**: Pengelompokan barang untuk kemudahan pencarian.
- **Transaksi Real-time**: Pencatatan transaksi penjualan yang cepat.
- **Manajemen User**: Pengaturan hak akses (Admin, Kasir, Bos).
- **Desain Premium**: Antarmuka responsif dengan Glassmorphism aesthetics.

## 🚀 Teknologi
- **Frontend**: React.js, Vite, Lucide React, Tailwind CSS / Vanilla CSS.
- **Backend**: Node.js, Express.js.
- **Database**: MariaDB / MySQL.
- **ORM**: Prisma.

## 🛠️ Instalasi

### 1. Clone Repositori
```bash
git clone https://github.com/arnofinformatika-bit/toko-sembako-arnof.git
cd toko-sembako-arnof
```

### 2. Setup Backend
```bash
npm install
npx prisma generate
node index.js
```

### 3. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```
